# homebridge-echonet-lite

Homebridge plugin for ECHONET Lite devices.

## Status

Implemented:

* Light bulb
* Air conditioner

Tested on:

* MoekadenRoom (I don't yet have an EchonetLite device)

## Usage

```js
"platforms": [
  {
    "platform": "ELPlatform",
    "enableRefreshSwitch": true
  }
]
```
